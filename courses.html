Course Descriptions

The following courses represent completed academic work within the Information Technology program.

First Semester

ACA 111 – College Student Success

General Education | Academic development and student success strategies.

CTI 110 – IT Foundations

Information Technology | Introduction to core IT concepts and computing systems.

CTS 115 – Information Systems Business Concepts

Information Systems | Business applications of technology and system fundamentals.

ENG 111 – Writing and Inquiry

English Composition | Academic writing, research, and critical thinking.

WEB 111 – Introduction to Web Graphics

Web Development | Digital design, image editing, and graphic composition.

No Retreat No Surrender – Typography Composition

Basic Adjustments – Color Correction Project

Second Semester

CSC 134 - C++ Programming

Computer Programming | Introduction to C++ programming language.

CSC 151 - JAVA Programming

Computer Programming | Introduction to JAVA programming language with object-oriented programming principles.

CTS 120 - Hardware/Software Support

Computer Information Technology | Covers the basic hardware of a personal computer, including installation, operations and interactions with software.

ENG 112 - Writing and Research in the Disciplines

English Composition | Introduces research techniques, documentation styles, and writing strategies.

Third Semester

CSC 121 - Python Programming

Computer Programming | Introduces computer programming using the Python programming language.

CTI 120 - Network and Security Foundations

Computer Technology Integration | Introduces students to the Network concepts, including networking terminology and protocols, local and wide area networks, and network standards.

CTS 285 - Systems Analysis & Design

Computer Information Technology | Introduces established and evolving methodologies for the analysis, design, and development of an information system.

Fourth Semester

CSC 251 - Advanced JAVA Programming

Computer Programming | Continuation of CSC 251 using the JAVA programming language with object-oriented programming principles.

DBA 120 - Database Programming 1

Database Management | Designed to develop SQL programming proficiency. Emphasis placed on data definition, data manipulation, and data control statements as well as on report generation.

WEB 115 - Web Markup and Scripting

Web Development | Introduces Worldwide Web Consortium (W3C) Internet programming using JavaScript. Topics include basic syntax, object-oriented programming, functions, variables, events, arrays, validation, accessibility, and web standards.

WEB 182 - PHP Programming

Web Development | Introduces students to the server-side, HTML-embedded scripting language PHP. Emphasis is placed on programming techniques required to create dynamic web pages.

Fifth Semester

CSC 227 - Cloud Application Development

Computer Science | Introduces how to build, deploy, host, and manage applications using cloud technologies.

CSC 289 - Programming Capstone Project

Comupter Science | Provides an opportunity to complete a significant programming project from the design phase through implementation with minimal instructor support.

WEB 151 - Mobile Application Development 1

Web Development | Introduces students to programming techniques, design, and development related to mobile applications. Topics include accessing device capabilities, compliance with industry standards, and programming for mobile applications.

Major Electives

NOS 120 - Linux Single User

Provides students with GUI and command line skills for using a Linux machine. Topics include Linux file system and access permissions, desktop or shell environments, command line editors, display servers, expression pattern matching, I/O redirection, and networking.
